# Phase 5 配線図 — 自走式自動開閉ごみ箱（全統合）

Phase 4 の配線に SG90 サーボ（フタ駆動）と制御系電源（9V電池）を追加した最終構成。

## 部品

- Arduino Uno
- VS1838B IR受信モジュール ＋ IRリモコン
- HC-SR04 超音波センサ（前方に向けて設置）
- SG90 サーボモータ（フタ駆動）
- DRV8835 デュアルモータドライバ
- DCモータ × 2（左右）＋ タイヤ、キャスター
- 電池ボックス（単3×4本 / 約6V）— モータ専用電源
- 9V電池 ＋ DCジャック（または5Vモバイルバッテリー→USB）— Arduino用電源

## 必要ライブラリ

- **IRremote v4.x**（ライブラリマネージャで「IRremote」をインストール）
- Servo（Arduino IDE 標準）

## ピン接続

### Arduino → VS1838B

| Arduino ピン | VS1838B ピン | 説明 |
|---|---|---|
| D2 | OUT | 受信信号 |
| 5V | VCC | 電源 |
| GND | GND | グラウンド |

### Arduino → HC-SR04

| Arduino ピン | HC-SR04 ピン | 説明 |
|---|---|---|
| D12 | TRIG | 超音波送信トリガ |
| D11 | ECHO | 超音波受信エコー |
| 5V | VCC | センサ電源 |
| GND | GND | グラウンド |

### Arduino → SG90 サーボ

| Arduino ピン | SG90 線 | 説明 |
|---|---|---|
| D9 | 信号線（オレンジ） | PWM 制御 |
| 5V | 電源（赤） | サーボ電源 |
| GND | GND（茶/黒） | グラウンド |

### Arduino → DRV8835

| Arduino ピン | DRV8835 ピン | 説明 |
|---|---|---|
| D7 | AIN1 | 左モータ 前進 |
| D6 | AIN2 | 左モータ 後退（PWM） |
| D5 | BIN2 | 右モータ 前進（PWM） |
| D4 | BIN1 | 右モータ 後退 |
| 5V | VCC | ロジック電源 |
| 5V | MODE | IN/INモード選択（HIGHに固定） |
| GND | GND | 共通グラウンド |

### DRV8835 → モータ・電源

| DRV8835 ピン | 接続先 | 説明 |
|---|---|---|
| AOUT1 | 左モータ + 端子 | |
| AOUT2 | 左モータ − 端子 | |
| BOUT1 | 右モータ + 端子 | |
| BOUT2 | 右モータ − 端子 | |
| VM | 電池ボックス + | モータ用電源（4.5〜6V） |
| GND | 電池ボックス − / Arduino GND | 共通グラウンド |

## 電源系統

```
[9V電池] ──── Arduino DCジャック（Vin）──→ 5V ──→ IR受信・HC-SR04・サーボ・DRV8835ロジック
[単3×4本（約6V）] ──→ DRV8835 VM ──→ DCモータ
        └──────────→ GND ←──────── Arduino GND（必ず共通にする）
```

```

  D6 ─────────────── AIN2
  D5 ─────────────── BIN2
  D4 ─────────────── BIN1

                     BOUT2 ── [右モータ −]
```

## 動作

1. リモコンの ▲▼◀▶ を**押している間だけ**前進・後退・左右旋回（離すと約0.2秒で停止）。
2. 前進中、前方 30cm 以内にユーザーを検知すると自動停止し、手かざし待ちになる。
3. 手を 5cm 以内にかざすとフタが開き、3秒後に自動で閉まる。
4. 手かざしが30秒なければ待機解除。OK ボタンでいつでも即時停止。

## 注意事項

- `IR_FWD` 等のボタンコードは **3-ir-test の実測値に書き換える**こと。
- フタ開閉中は走行しない設計（走行コマンドを受けるとフタを閉じてから動く）。
- SG90 は Arduino 5V から給電するが、9V電池駆動時はレギュレータの容量が小さいため、サーボ動作時にリセットする場合は5Vモバイルバッテリー給電に切り替える。
- HC-SR04 は車体前面の、フタ開閉時に手をかざしやすい高さに取り付ける。
- Arduino の電源とモータ電源は**必ず GND を共通**にする。
