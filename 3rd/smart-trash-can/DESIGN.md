# 音で呼べる自動開閉ごみ箱 — 設計書

## 1. 概要

手を叩くとごみ箱がその方向に自走し、近づいたら自動でフタが開閉するシステム。

**動作フロー**

```
拍手検知 → 方向判定 → 旋回・走行 → 距離監視 → 停止 → 手かざし検知 → フタ開閉
```

## 4. マイク配置と方向判定

### 4.1 配置

ごみ箱**本体**の上部側面に4つのマイクを90°間隔で配置する。  
フタ上への設置は禁止（蓋開閉でマイク方向が変わるため）。

```
        マイク A（0°/前方）
            ↑
    [D] ←  [ ]  → [B]
   (270°/右)    (90°/左)
            ↓
        マイク C（180°/後方）
```

### 4.2 拍手検知アルゴリズム

拍手は「短時間の大振幅バースト」として定義し、環境ノイズや話し声と区別する。

**判定条件（AND）**
1. いずれかのチャンネルのピーク ADC 値が `MIC_THRESHOLD`（例：600）を超える
2. ピーク持続時間が `CLAP_MAX_MS`（例：150ms）以内に収まる（長い音はノイズと判定）

**拍手パターン定義**

| パターン | 意味 |
|---|---|
| 拍手（単発・連続どちらでも可） | ごみ箱を呼ぶ／走行中の方向補正ビーコン |

拍手を検知するたびに即座に方向推定へ使う。連続して叩くほど方向精度が上がり追尾がスムーズになる。

### 4.3 方向推定アルゴリズム

単発1回では誤差が大きいため、`DIR_SAMPLES`（例：3）回の拍手ベクトルを平均して方向を確定する。  
`DETECT_TIMEOUT_MS`（例：10000ms）以内に `DIR_SAMPLES` 回検知できなければ IDLE に戻る。

**1回の拍手から角度を推定する手順**

1. 4chピーク取得：`pA, pB, pC, pD`
2. X成分・Y成分に分解：
   ```
   Vx = pA - pC   // 前後軸（前が正）
   Vy = pB - pD   // 左右軸（左が正）
   ```
3. 角度算出：`θ = atan2(Vy, Vx)`（rad → deg変換）

4. `DIR_SAMPLES` 回のベクトルを累積平均してノイズ低減：
   ```
   Vx_avg += Vx;  Vy_avg += Vy;
   θ_final = atan2(Vy_avg, Vx_avg)
   ```

### 4.4 方向→旋回角度の対応

`θ_final` から旋回量を直接算出することで、離散テーブルよりも滑らかに対応する。

```
旋回時間 t = TIME_180DEG * (|θ_final| / 180)
旋回方向  = θ_final > 0 ? 左旋回 : 右旋回
```

> **符号の根拠**: Vy = pB - pD（B=左）なので左から音が来ると θ > 0。左に向くには反時計回り（左旋回）が必要。

## 5. 駆動制御

### 5.1 差動駆動

| 動作 | 左モータ | 右モータ |
|---|---|---|
| 前進 | 前進 | 前進 |
| 右旋回（時計回り） | 前進 | 停止または後退 |
| 左旋回（反時計回り） | 停止または後退 | 前進 |
| 停止 | 停止 | 停止 |

### 5.2 旋回時間による角度制御

エンコーダなしのため、モータ駆動時間で旋回角度を近似する。  
事前キャリブレーションで「180°旋回に要する時間（ms）」を実測し定数として定義する。

```cpp
// 例: 180°旋回 = 800ms（実測値に要調整）
const int TIME_180DEG = 800;
```

旋回角度 θ に対する駆動時間：

```
t = TIME_180DEG * (θ / 180)
```

### 5.3 走行中の方向補正

APPROACHING 中も拍手を検知するたびに以下の手順で軌道を修正する（ビーコンモード）。

1. 一時停止（モータ停止）
2. 拍手1回で θ を即時推定（DIR_SAMPLES は使わず単発で補正）
3. |θ| が `CORRECT_THRESH`（例：20°）以上なら旋回して向きを修正
4. 前進を再開

停止→補正→再開をワンセットにすることで、差動走行中のノイズ混入を避ける。

### 5.4 走行停止条件

超音波センサで前方距離を監視し、`STOP_DIST`（例：20cm）以下になったら即停止。

### 5.5 停止後の最終整列（ALIGNING）

APPROACHING 最後に蓄積された θ_final の残余角（旋回後に残ったズレ）を使い、音源を待たずに低速旋回して向きを補正する。

```
// 最後の θ_final がそのまま残余角として使える
旋回時間 t_align = TIME_180DEG * (|θ_final| / 180) * ALIGN_SPEED_RATIO
```

新たに拍手が来た場合はその方向で θ を上書きして旋回量を更新する。  
`ALIGN_TIMEOUT_MS`（例：3000ms）以内に完了しない場合は現在向きのまま WAITING に移行する。

```
マイクA方向 = 蓋が開く方向（設置時の固定制約）
```

## 6. フタ開閉制御

### 6.1 手かざし検知

走行停止後、同じ HC-SR04 センサでフタ前方の距離を監視する。  
`HAND_DIST`（例：15cm）以内に手を検知したらフタを開ける。  
`WAITING_TIMEOUT_MS`（例：30000ms）以内に手かざしがなければ IDLE に戻る。

### 6.2 サーボ動作

| 状態 | サーボ角度 |
|---|---|
| 閉 | 0° |
| 全開 | 180° |

サーボをスイング蓋の回転軸付近に設置し、アームで蓋の後端を直接押し開く構造。  
軸近傍への設置により小トルク（SG90：約1.8kg·cm）で開閉可能。  
開状態は `OPEN_DURATION`（例：3000ms）のタイムアウトで自動閉鎖する。

## 7. 電源設計

```
[9V電池] → Arduino Vin → 5V レギュレータ → センサ・サーボ
[単3×4本（6V）電池ボックス] → DRV8835 VM → DCモータ
                            ↘ GND共通 → Arduino GND
```

- 制御系とモータ系の GND は必ず共通にする。
- モータ動作時の電圧降下がセンサに影響しないよう電源を分離する。

## 8. ステートマシン

```
[IDLE]
  ↓ 拍手検知
[DETECTING]   ← DIR_SAMPLES 回の拍手ベクトルを平均して θ_final を確定
  ↓ 方向確定
[TURNING]     ← θ_final に基づき時間制御で旋回
  ↓ 旋回完了
[APPROACHING] ← 前進しながら超音波で距離監視
              ← 拍手検知のたびに一時停止→方向再推定→軌道修正（ビーコン追尾）
  ↓ 距離 ≤ STOP_DIST
[ALIGNING]    ← θ_final の残余角をもとに低速旋回して蓋をユーザー方向に向ける
  ↓ 旋回完了（または ALIGN_TIMEOUT_MS 超過）
[WAITING]     ← 手かざし待機（超音波で手を検知）
  ↓ 距離 ≤ HAND_DIST
[OPEN]        ← フタ開（サーボ 180°）
  ↓ OPEN_DURATION 経過
[CLOSE]       ← フタ閉（サーボ 0°）
  ↓ 完了
[IDLE]

※ DETECTING は DETECT_TIMEOUT_MS 以内に DIR_SAMPLES 回未達なら IDLE に戻る
※ ALIGNING は ALIGN_TIMEOUT_MS 超過で現在向きのまま WAITING に移行
※ WAITING は WAITING_TIMEOUT_MS 以内に手かざしなければ IDLE に戻る
```

## 10. 主要パラメータ一覧

| 定数名 | 初期値 | 説明 |
|---|---|---|
| `MIC_THRESHOLD` | 600 | 拍手判定の ADC しきい値（0〜1023） |
| `CLAP_MAX_MS` | 150 | 拍手とみなすバースト最大持続時間（ms）。これ以上長い音はノイズ扱い |
| `DIR_SAMPLES` | 3 | 方向確定に使う拍手検知回数（多いほど精度向上） |
| `DETECT_TIMEOUT_MS` | 10000 | DETECTING でのタイムアウト（ms）。超過で IDLE 復帰 |
| `CORRECT_THRESH` | 20 | APPROACHING 中の補正を行う最小ズレ角度（°）。これ未満は補正しない |
| `ALIGN_SPEED_RATIO` | 0.5 | ALIGNING 時の旋回速度倍率（TURN_SPEED に対する比） |
| `ALIGN_TIMEOUT_MS` | 3000 | ALIGNING でのタイムアウト（ms）。超過で WAITING 移行 |
| `WAITING_TIMEOUT_MS` | 30000 | WAITING でのタイムアウト（ms）。手かざしなしで IDLE 復帰 |
| `SAMPLE_MS` | 50 | マイクサンプリング時間（ms） |
| `TIME_180DEG` | 800 | 180°旋回の駆動時間（ms）※要実測 |
| `DRIVE_SPEED` | 180 | 前進時の PWM 値（0〜255） |
| `TURN_SPEED` | 150 | 旋回時の PWM 値（0〜255） |
| `STOP_DIST` | 20 | 走行停止距離（cm） |
| `HAND_DIST` | 15 | 手かざし検知距離（cm） |
| `OPEN_DURATION` | 3000 | フタ開放時間（ms） |
| `ANGLE_OPEN` | 180 | フタ全開のサーボ角度（°） |
| `ANGLE_CLOSED` | 0 | フタ閉のサーボ角度（°） |
