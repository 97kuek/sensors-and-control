# 音で呼べる自動開閉ごみ箱 — 設計書

## 1. 概要

手を叩くとごみ箱がその方向に自走し、近づいたら自動でフタが開閉するシステム。

**動作フロー**

```
拍手検知 → 方向判定 → 旋回・走行 → 距離監視 → 停止 → 手かざし検知 → フタ開閉
```

## 4. マイク配置と方向判定

### 4.1 配置

ごみ箱上面（またはフタ外周）に3つのマイクを120°間隔で配置する。

```
        マイク A（0°/前方）
            ↑
           [A]
          /   \
        [C]   [B]
  (240°/右後) (120°/左後)
```

### 4.2 拍手検知アルゴリズム

1. 3チャンネルを短時間（例：50ms）連続サンプリングし、各チャンネルのピーク電圧を記録する。
2. 最大ピークがしきい値（例：ADC値 600/1023）を超えたら拍手と判定する。
3. 3チャンネルのピーク比較で最大チャンネルを特定し、方向（前/左後/右後）を決定する。
4. 前後左右の中間方向は2チャンネルのピーク差の比率で補間する。

### 4.3 方向→旋回角度の対応

| 最大チャンネル | 判定方向 | 旋回動作 |
|---|---|---|
| A のみ大 | 前方（±30°以内） | 旋回なし、即前進 |
| B > C | 左後方（90°〜180°） | 右旋回 |
| C > B | 右後方（180°〜270°） | 左旋回 |
| A と B が同等 | 左前方（30°〜90°） | 右旋回（小） |
| A と C が同等 | 右前方（270°〜330°） | 左旋回（小） |
| B と C が同等 | 真後ろ（180°） | 旋回（大） |

## 5. 駆動制御

### 5.1 差動駆動

| 動作 | 左モータ | 右モータ |
|---|---|---|
| 前進 | 前進 | 前進 |
| 右旋回（時計回り） | 前進 | 停止または後退 |
| 左旋回（反時計回り） | 停止または後退 | 前進 |
| 停止 | 停止 | 停止 |

### 5.2 旋回時間による角度制御

エンコーダなしのため、モータ駆動時間で旋回角度を近似する。  
事前キャリブレーションで「180°旋回に要する時間（ms）」を実測し定数として定義する。

```cpp
// 例: 180°旋回 = 800ms（実測値に要調整）
const int TIME_180DEG = 800;
```

旋回角度 θ に対する駆動時間：

```
t = TIME_180DEG * (θ / 180)
```

### 5.3 走行停止条件

超音波センサで前方距離を監視し、`STOP_DIST`（例：20cm）以下になったら即停止。

---

## 6. フタ開閉制御

### 6.1 手かざし検知

走行停止後、同じ HC-SR04 センサでフタ前方の距離を監視する。  
`HAND_DIST`（例：15cm）以内に手を検知したらフタを開ける。

### 6.2 サーボ動作

| 状態 | サーボ角度 |
|---|---|
| 閉 | 0° |
| 全開 | 180° |

サーボのアームをフタ裏面に固定し、回転でスイング蓋を押し開く構造。  
開状態は `OPEN_DURATION`（例：3000ms）のタイムアウトで自動閉鎖する。

---

## 7. 電源設計

```
[9V電池] → Arduino Vin → 5V レギュレータ → センサ・サーボ
[単3×4本（6V）電池ボックス] → DRV8835 VM → DCモータ
                            ↘ GND共通 → Arduino GND
```

- 制御系とモータ系の GND は必ず共通にする。
- モータ動作時の電圧降下がセンサに影響しないよう電源を分離する。

## 8. ステートマシン

```
[IDLE]
  ↓ 拍手検知
[DETECTING]   ← 方向判定（50ms サンプリング）
  ↓ 方向確定
[TURNING]     ← 旋回（時間制御）
  ↓ 旋回完了
[APPROACHING] ← 前進しながら超音波で距離監視
  ↓ 距離 ≤ STOP_DIST
[WAITING]     ← 手かざし待機（超音波で手を検知）
  ↓ 距離 ≤ HAND_DIST
[OPEN]        ← フタ開（サーボ 180°）
  ↓ OPEN_DURATION 経過
[CLOSE]       ← フタ閉（サーボ 0°）
  ↓ 完了
[IDLE]
```

## 10. 主要パラメータ一覧

| 定数名 | 初期値 | 説明 |
|---|---|---|
| `MIC_THRESHOLD` | 600 | 拍手判定の ADC しきい値（0〜1023） |
| `SAMPLE_MS` | 50 | マイクサンプリング時間（ms） |
| `TIME_180DEG` | 800 | 180°旋回の駆動時間（ms）※要実測 |
| `DRIVE_SPEED` | 180 | 前進時の PWM 値（0〜255） |
| `TURN_SPEED` | 150 | 旋回時の PWM 値（0〜255） |

| `HAND_DIST` | 15 | 手かざし検知距離（cm） |
| `OPEN_DURATION` | 3000 | フタ開放時間（ms） |
| `ANGLE_OPEN` | 180 | フタ全開のサーボ角度（°） |
| `ANGLE_CLOSED` | 0 | フタ閉のサーボ角度（°） |
