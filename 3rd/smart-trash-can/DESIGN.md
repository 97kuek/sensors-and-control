# IRリモコンで操作できる自動開閉ごみ箱 — 設計書

## 1. 概要

IRリモコンのボタンで前後左右を手動操作してごみ箱を呼び寄せ、手をかざすとフタが自動で開閉するシステム。

### 動作フロー

```text
リモコン操作（前後左右） → モーター制御 → 走行 → 停止 → 手かざし → フタ開閉
```

## 4. リモコン操作

### 4.1 ボタン割り当て

スターターキット付属のIRリモコンのボタンに以下の動作を割り当てる。  
使用するボタンコードは事前に `IRrecv.decode()` で実測して定数化する。

| ボタン | 動作 |
| --- | --- |
| ▲（上） | 前進 |
| ▼（下） | 後退 |
| ◀（左） | 左旋回 |
| ▶（右） | 右旋回 |
| OK / 決定 | 停止 |

### 4.2 受信の仕組み

VS1838BをD2に接続し、IRremoteライブラリでコマンドを受信する。  
受信したコードをボタン定数と照合して対応する動作を実行する。  
ボタンを離した後の停止は `STOP` コードの受信、またはタイムアウトで行う。

### 5.1 差動駆動

| 動作 | 左モータ | 右モータ |
| --- | --- | --- |
| 前進 | 前進 | 前進 |
| 後退 | 後退 | 後退 |

| 左旋回（反時計回り） | 後退 | 前進 |
| 停止 | 停止 | 停止 |

## 6. フタ開閉制御

### 6.1 手かざし検知

走行停止後、同じHC-SR04でフタ前方の距離を監視する。  
`HAND_DIST`（例：5cm）以内に手を検知したらフタを開ける。  
`WAITING_TIMEOUT_MS`（例：30000ms）以内に手かざしがなければIDLEに戻る。

### 6.2 サーボ動作

| 状態 | サーボ角度 |
| --- | --- |
| 閉 | 40° |
| 全開 | 180° |

開状態は `OPEN_DURATION`（例：3000ms）のタイムアウトで自動閉鎖する。

## 7. 電源設計

```text
[9V電池] → Arduino Vin → 5Vレギュレータ → センサ・サーボ・IR受信
[単3×4本（6V）電池ボックス] → DRV8835 VM → DCモータ
                            ↘ GND共通 → Arduino GND
```

- 制御系とモータ系のGNDは必ず共通にする。
- モータ動作時の電圧降下がセンサに影響しないよう電源を分離する。

---

## 8. ステートマシン

```text
[IDLE]
  ↓ IRコマンド受信（前後左右）
[MOVING]      ← ボタンに対応したモーター制御を実行
  ↓ STOPコマンド受信
[WAITING]     ← 手かざし待機（HC-SR04 で手を検知）
              ← WAITING_TIMEOUT_MS 超過で IDLE に戻る
  ↓ HAND_DIST 以内に手を検知
[OPEN]        ← フタ開（サーボ 180°）
  ↓ OPEN_DURATION 経過
[CLOSE]       ← フタ閉（サーボ 40°）
  ↓ 完了
[IDLE]
```

## 10. 主要パラメータ一覧

| 定数名 | 初期値 | 説明 |
| --- | --- | --- |
| `IR_FWD` | 実測値 | 前進ボタンのコード |
| `IR_BWD` | 実測値 | 後退ボタンのコード |
| `IR_LEFT` | 実測値 | 左旋回ボタンのコード |
| `IR_RIGHT` | 実測値 | 右旋回ボタンのコード |
| `IR_STOP` | 実測値 | 停止ボタンのコード |
| `HAND_DIST` | 5 | 手かざし検知距離（cm） |
| `WAITING_TIMEOUT_MS` | 30000 | WAITINGでのタイムアウト（ms） |
| `OPEN_DURATION` | 3000 | フタ開放時間（ms） |
| `DRIVE_SPEED` | 180 | 前進・後退時のPWM値（0〜255） |
| `TURN_SPEED` | 150 | 旋回時のPWM値（0〜255） |
| `ANGLE_OPEN` | 180 | フタ全開のサーボ角度（°） |
| `ANGLE_CLOSED` | 40 | フタ閉のサーボ角度（°） |
