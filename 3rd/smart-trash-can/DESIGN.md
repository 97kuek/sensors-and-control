# 自走式自動開閉ごみ箱 — 設計書

## 1. 概要

IRリモコンの方向ボタンを**押している間だけ**走行し、離すと停止するラジコン式操作でごみ箱を呼び寄せる。前進中はHC-SR04でユーザーとの距離を監視し、一定距離以内に接近すると**自動で停止**。停止後は同じHC-SR04で手かざしを検知し、サーボでフタを自動開閉する。

### 動作フロー

```text
ボタン長押し → 走行（押している間だけ） → ユーザーに接近 → 自動停止 → 手かざし → フタ開閉
```

## 4. リモコン操作

### 4.1 ボタン割り当て

スターターキット付属のIRリモコンのボタンに以下の動作を割り当てる。  
使用するボタンコードは事前に `IRrecv.decode()` で実測して定数化する。

| ボタン | 動作 |
| --- | --- |
| ▲（上） | 押している間：前進 |
| ▼（下） | 押している間：後退 |
| ◀（左） | 押している間：左旋回 |
| ▶（右） | 押している間：右旋回 |
| OK / 決定 | 即時停止（保険用） |

### 4.2 長押し（押している間だけ動作）の仕組み

NECフォーマットのIRリモコンは、ボタンを押し続けている間およそ110ms間隔で**リピートコード**（`0xFFFFFFFF` / IRremoteではrepeatフラグ）を送信し続ける。これを利用して「押している間だけ動作・離すと停止」を実現する。

```text
1. 方向ボタンのコードを受信 → 対応する走行動作を開始し、lastIrTime を更新
2. リピートコードを受信 → lastIrTime を更新（動作継続）
3. millis() - lastIrTime > IR_REPEAT_TIMEOUT_MS → ボタンが離されたと判定して停止
```

- `IR_REPEAT_TIMEOUT_MS` はリピート間隔（約110ms）より十分長く、かつ離してから停止までの遅れが気にならない値（150〜200ms程度）に設定する。
- リピートコードは「直前に受信した方向ボタンの継続」として扱う。

## 5. 駆動制御

### 5.1 差動駆動

| 動作 | 左モータ | 右モータ |
| --- | --- | --- |
| 前進 | 前進 | 前進 |
| 後退 | 後退 | 後退 |
| 右旋回（時計回り） | 前進 | 後退 |
| 左旋回（反時計回り） | 後退 | 前進 |
| 停止 | 停止 | 停止 |

### 5.2 接近時の自動停止

**前進中のみ**、前方に設置したHC-SR04でユーザーとの距離を周期的に計測する。  
`STOP_DIST`（例：30cm）以内に接近したら、ボタンが押されていても自動で停止し、手かざし待機（WAITING）に移行する。

- 後退・旋回中はセンサが進行方向を向かないため、自動停止は行わない。
- 計測は `SONAR_INTERVAL_MS`（例：60ms）周期とし、走行制御をブロックしない。

---

## 6. フタ開閉制御

### 6.1 手かざし検知

自動停止後、同じHC-SR04でフタ前方の距離を監視する。  
`HAND_DIST`（例：5cm）以内に手を検知したらフタを開ける。  
`WAITING_TIMEOUT_MS`（例：30000ms）以内に手かざしがなければIDLEに戻る。

### 6.2 サーボ動作

| 状態 | サーボ角度 |
| --- | --- |
| 閉 | 40° |
| 全開 | 160°（Phase 2 で実機調整済み） |

開状態は `OPEN_DURATION`（例：3000ms）のタイムアウトで自動閉鎖する。

---

## 7. 電源設計

```text
[9V電池] → Arduino Vin → 5Vレギュレータ → センサ・サーボ・IR受信
[単3×4本（6V）電池ボックス] → DRV8835 VM → DCモータ
                            ↘ GND共通 → Arduino GND
```

- 制御系とモータ系のGNDは必ず共通にする。
- モータ動作時の電圧降下がセンサに影響しないよう電源を分離する。

## 8. ステートマシン

```text
[IDLE]
  ↓ 方向ボタン受信
[MOVING]      ← 押している間だけ走行（リピートコードで継続）
  │
  ├─ ボタン解放（IR_REPEAT_TIMEOUT_MS 途絶）→ [IDLE]
  │
  ↓ 前進中に距離 ≤ STOP_DIST（自動停止）
[WAITING]     ← 手かざし待機（HC-SR04 で手を検知）
              ← WAITING_TIMEOUT_MS 超過で IDLE に戻る
  ↓ HAND_DIST 以内に手を検知
[OPEN]        ← フタ開（サーボ 160°）
  ↓ OPEN_DURATION 経過
[CLOSE]       ← フタ閉（サーボ 40°）
  ↓ 完了
[IDLE]
```

- WAITING・OPEN・CLOSE 中に方向ボタンを受信した場合はフタを閉じてから MOVING に移行する（誤動作防止のため OPEN 中は走行しない）。

## 9. 実装ステップ

| フェーズ | フォルダ | 内容 | 状態 |
| --- | --- | --- | --- |
| Phase 1 | `1-motor-test/` | DCモータ単体テスト（前進・旋回・停止） | 完了 |

## 10. 主要パラメータ一覧

| 定数名 | 初期値 | 説明 |
| --- | --- | --- |
| `IR_FWD` | 実測値 | 前進ボタンのコード |
