# Phase 1 配線図 — DCモータテスト（DRV8835）

## 部品

- Arduino Uno
- DRV8835 デュアルモータドライバ
- DCモータ × 2（左右）
- 電池ボックス（単3×4本 / 約6V）— モータ専用電源
- Arduino 用電源（USB または 9V 電池）

---

## ピン接続

### Arduino → DRV8835

| Arduino ピン | DRV8835 ピン | 説明 |
|---|---|---|
| D7 | AIN1 | 左モータ 前進 |
| D6 | AIN2 | 左モータ 後退（PWM） |
| D5 | BIN1 | 右モータ 前進（PWM） |
| D4 | BIN2 | 右モータ 後退 |
| 5V | VCC | ロジック電源 |
| 5V | MODE | IN/INモード選択（HIGHに固定） |
| GND | GND | 共通グラウンド |

### DRV8835 → モータ・電源

| DRV8835 ピン | 接続先 | 説明 |
|---|---|---|
| AOUT1 | 左モータ + 端子 | |
| AOUT2 | 左モータ − 端子 | |
| BOUT1 | 右モータ + 端子 | |
| BOUT2 | 右モータ − 端子 | |
| VM | 電池ボックス + | モータ用電源（4.5〜6V） |
| GND | 電池ボックス − / Arduino GND | 共通グラウンド |

---

## 配線図（概略）

```
[Arduino]              [DRV8835]
  D7 ──────────────── AIN1
  D6 ──────────────── AIN2
  D5 ──────────────── BIN1
  D4 ──────────────── BIN2
  GND ─────────────── GND ──── [電池ボックス −]

                      AOUT1 ── [左モータ +]
                      AOUT2 ── [左モータ −]
                      BOUT1 ── [右モータ +]
                      BOUT2 ── [右モータ −]
                      VM ───── [電池ボックス +]
```

---

## 注意事項

- Arduino の電源（USB/Vin）とモータ電源（電池ボックス）は**必ず GND を共通**にする。
- モータが逆回転する場合は、そのモータの `+` / `−` 端子を入れ替える（コードは変えない）。
- DRV8835 の VM は最大 11V まで対応。単3×4本（約6V）で動作確認済み想定。
