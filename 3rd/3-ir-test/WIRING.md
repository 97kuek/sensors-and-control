# Phase 3 配線図 — IRリモコン受信テスト（VS1838B）

## 部品

- Arduino Uno
- VS1838B IR受信モジュール（または互換品）
- 市販IRリモコン（スターターキット付属等）
- Arduino 用電源（USB）

## 必要ライブラリ

- **IRremote v4.x** — Arduino IDE のライブラリマネージャで「IRremote」（作者: shirriff, z3t0, ArminJo）をインストール

---

## ピン接続

### Arduino → VS1838B

| Arduino ピン | VS1838B ピン | 説明 |
|---|---|---|
| D2 | OUT（DAT / S） | 受信信号 |
| 5V | VCC | 電源 |
| GND | GND | グラウンド |

> **注意:** VS1838B 単体（足3本）の場合、受光面（半球のドーム）を手前に見て **左から OUT・GND・VCC**。
> モジュール基板に実装されている場合はピン配置が異なることがあるので、基板のシルク印刷（S / − / VCC など）に従う。

```
[Arduino]              [VS1838B]
  D2 ──────────────── OUT
  5V ──────────────── VCC
  GND ─────────────── GND
```

## 使い方

1. スケッチを書き込み、シリアルモニタを 9600bps で開く。
2. リモコンの ▲▼◀▶・OK ボタンを順に押し、表示される `cmd=0x??` を記録する。
3. 長押しすると、離した時に `RELEASE` が表示されることを確認する（長押し処理の検証）。
4. 記録したコードを `4-drive-control` / `smart-trash-can` の `IR_FWD` 等の定数に転記する。

## 注意事項

- リモコンに絶縁シート（電池の保護フィルム)が挟まっている場合は抜いておく。
- 蛍光灯・直射日光の近くでは誤受信することがある。
- `PRESS` が連続で大量に出る場合はリモコンのフォーマットがNEC以外の可能性がある。その場合はスケッチ冒頭の `#define DECODE_NEC` を削除して全フォーマットでデコードし、表示される protocol を確認する。
