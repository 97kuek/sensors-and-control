# Phase 4 配線図 — IR長押し走行制御＋接近自動停止

Phase 1（DRV8835＋DCモータ）の配線に、IR受信モジュールと HC-SR04 を追加した構成。

## 部品

- Arduino Uno
- VS1838B IR受信モジュール ＋ IRリモコン
- HC-SR04 超音波センサ（前方に向けて設置）
- DRV8835 デュアルモータドライバ
- DCモータ × 2（左右）
- 電池ボックス（単3×4本 / 約6V）— モータ専用電源
- Arduino 用電源（USB または 9V 電池）

## 必要ライブラリ

- **IRremote v4.x**（ライブラリマネージャで「IRremote」をインストール）

---

## ピン接続

### Arduino → VS1838B

| Arduino ピン | VS1838B ピン | 説明 |
|---|---|---|
| D2 | OUT | 受信信号 |
| 5V | VCC | 電源 |
| GND | GND | グラウンド |

### Arduino → HC-SR04

| Arduino ピン | HC-SR04 ピン | 説明 |
|---|---|---|
| D12 | TRIG | 超音波送信トリガ |
| D11 | ECHO | 超音波受信エコー |
| 5V | VCC | センサ電源 |
| GND | GND | グラウンド |

### Arduino → DRV8835

| Arduino ピン | DRV8835 ピン | 説明 |
|---|---|---|
| D7 | AIN1 | 左モータ 前進 |
| D6 | AIN2 | 左モータ 後退（PWM） |
| D5 | BIN2 | 右モータ 前進（PWM） |
| D4 | BIN1 | 右モータ 後退 |
| 5V | VCC | ロジック電源 |
| 5V | MODE | IN/INモード選択（HIGHに固定） |
| GND | GND | 共通グラウンド |

### DRV8835 → モータ・電源

| DRV8835 ピン | 接続先 | 説明 |
|---|---|---|
| AOUT1 | 左モータ + 端子 | |
| AOUT2 | 左モータ − 端子 | |
| BOUT1 | 右モータ + 端子 | |
| BOUT2 | 右モータ − 端子 | |
| VM | 電池ボックス + | モータ用電源（4.5〜6V） |
| GND | 電池ボックス − / Arduino GND | 共通グラウンド |

## 配線図（概略）

```
[Arduino]            [VS1838B]
  D2 ─────────────── OUT
                     VCC ── 5V / GND ── GND

[Arduino]            [HC-SR04]（前方に向ける）
  D12 ────────────── TRIG
  D11 ────────────── ECHO
                     VCC ── 5V / GND ── GND

[Arduino]            [DRV8835]
  D7 ─────────────── AIN1
  D6 ─────────────── AIN2
  D5 ─────────────── BIN2
  D4 ─────────────── BIN1
  5V ─────────────── VCC, MODE
  GND ────────────── GND ──── [電池ボックス −]

                     AOUT1 ── [左モータ +]
                     AOUT2 ── [左モータ −]
                     BOUT1 ── [右モータ +]
                     BOUT2 ── [右モータ −]
                     VM ───── [電池ボックス +]
```

---

## 動作

1. リモコンの ▲▼◀▶ を**押している間だけ**前進・後退・左右旋回する。
2. ボタンを離すと約0.2秒以内に停止する。
3. 前進中に前方 30cm 以内に障害物（ユーザー）を検知すると自動停止する。
4. OK ボタンでいつでも即時停止できる。

## 注意事項

- `IR_FWD` 等のボタンコードは **3-ir-test の実測値に書き換える**こと（初期値はElegoo系リモコンの代表値）。
- モータが逆回転する場合は、そのモータの `+` / `−` 端子を入れ替える（コードは変えない）。
- 左右で前進・後退時のPWMのかけ方が異なる（左前進・右後退はスロー減衰駆動）。同じPWM値でも左右の速度が微妙に違う場合は `DRIVE_SPEED` を基準に直進するよう旋回成分を調整するのではなく、まず機械的な原因（タイヤ・ギア）を確認する。
- Arduino の電源とモータ電源は**必ず GND を共通**にする。
