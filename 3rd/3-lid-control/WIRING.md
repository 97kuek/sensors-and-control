# Phase 3 配線図 — 手かざし検知・フタ開閉（HC-SR04 + SG90）

## 部品

- Arduino Uno
- HC-SR04 超音波センサ
- SG90 サーボモータ
- Arduino 用電源（USB または 9V 電池）

---

## ピン接続

### Arduino → HC-SR04

| Arduino ピン | HC-SR04 ピン | 説明 |
|---|---|---|
| D12 | TRIG | 超音波送信トリガ |
| D11 | ECHO | 超音波受信エコー |
| 5V | VCC | センサ電源 |
| GND | GND | グラウンド |

### Arduino → SG90 サーボ

| Arduino ピン | SG90 線 | 説明 |
|---|---|---|
| D9 | 信号線（オレンジ） | PWM 制御 |
| 5V | 電源（赤） | サーボ電源 |
| GND | GND（茶/黒） | グラウンド |

---

## 配線図（概略）

```
[Arduino]              [HC-SR04]
  D12 ─────────────── TRIG
  D11 ─────────────── ECHO
  5V ──────────────── VCC
  GND ─────────────── GND

[Arduino]              [SG90]
  D9 ──────────────── 信号線（オレンジ）
  5V ──────────────── 電源（赤）
  GND ─────────────── GND（茶/黒）
```

---

## 動作

1. 電源投入時：フタ閉（40°）
2. 手を 5cm 以内に近づける → フタ開（180°）
3. 3 秒後に自動でフタ閉（40°）

## 注意事項

- SG90 は Arduino の 5V から直接給電可能（1 個のみ）。
- HC-SR04 の ECHO は 5V 出力なので D11（デジタル入力）に直結で問題なし。
- サーボの取り付け角度によっては `ANGLE_OPEN` / `ANGLE_CLOSED` の値を調整する。
